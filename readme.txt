Connect AutomatorWP with LearnDash

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[LearnDash](https://www.learndash.com/ "LearnDash") is a full-featured WordPress LMS plugin that allows you to easily create and sell courses online.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with LearnDash adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User views any/specific quiz.
* User views any/specific lesson.
* User views any/specific topic.
* User views any/specific course.
* User attempts, passes or fails any/specific quiz.
* User completes any/specific quiz with a grade percentage greater than, less than or equal to a specific percentage.
* User completes any/specific topic.
* User completes any/specific lesson.
* User completes any/specific course.

= Actions =

* Mark topic as completed.
* Mark lesson as completed.
* Mark course as completed.
* Enroll user to all/specific course.
* Create a group.
* Add user to a group.
* Make user the leader of a group.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/learndash/ "AutomatorWP - LearnDash") available that let's you expand the integration between AutomatorWP and LearnDash bringing you more ways to automatize your elearning website.

= What is AutomatorWP? =

AutomatorWP is an automator plugin that lets you connect your WordPress plugins together and add automation to unlimited workflows.

You can automate anything including sales, marketing, administrative tasks, learning and any other kind of processes you want letting you save time and get focused on your most important work.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
